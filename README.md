## Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/)
-   [npm](https://www.npmjs.com/)

### Run as Docker container

Run the application:

Build, Run and Stop the Docker container:

```bash
make build
make up
make down
```

### Development

Run the application locally:

```bash
git clone https://github.com/mahmudulmurad/scheduling_app.git
cd scheduling_app
npm install
npm start
```
